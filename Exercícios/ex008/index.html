Formatação de Texto

Negrito

Vou colocar uma parte do texto em negrito, utilizando a tag Strong. A tag STRONG é para dizer que a parte do texto é forte.

Itálico

Colocando em itálico utilizando a tag EM, nessa parte. Sempre optar por tags que tenham significados(semântica). A tag EM significa ênfase.

Marcar um texto e mudar a sua cor

Para marcar um texto, como se fosse um marca texto, usa-se a tag MARK. Exemplo: Esse texto está com a tag Mark e com Style(CSS) para mudar a cor do texto.
Para isso eu vou usar um seletor para a tag style, para não precisar mudar a cor de cada mark.

Texto grande e texto pequeno

Para deixar um texto grande utiliza-se a tag big. Para deixar um texto pequeno usa-se a tag small.

Texto Deletado

Para deixar um texto como se fosse riscado, para indicar que pode ser lido, mas não deve ser considerado.

Texto Inserido

Utilizar a tag ins para dar destaque ao texto

Texto Sobrescrito

É o elevado de uma equação, por exemplo: x10 + 20x + 3

Texto Subscrito

Um bom exemplo são compostos químicos, exemplo: H2O
